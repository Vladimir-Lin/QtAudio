開發版本（Beta）

目前的Beta版已經實作完整，所有音訊驅動介面均可成功編譯，但並未全部通過完整測試，細部依然有相當數量需要修正。

已經確認可正常播放及錄音的音訊驅動介面 :
  • Windows Direct Sound（可錄音、播放、輸出音量控制，穩定性甚高。）
  • Windows Multimedia Extensions（可錄音、播放、輸出音量控制，穩定性甚高。）
  • Windows Audio Session API（可錄音、播放，穩定性甚高。）
  • Windows Driver Model - Kernel Streaming（可播放，錄音不甚穩定。）
  • Advanced Linux Sound Architecture（可播放、錄音，不穩定，細節依然需要修正）
  • JACK Audio Connection Kit（可播放，不穩定，細節依然需要修正）
  • Open Sound System（可播放、錄音，不穩定，細節依然需要修正）
  • AudioScience HPI（無法測試，但可以編譯）
  • Apple Core Audio（可錄音、播放、輸出音量控制， 穩定性甚高。）
  • FFmpeg（可壓縮、解壓縮、重新取樣，穩定性甚高。）

已經測試的列表可以在這裡找到《音訊驅動介面支援狀態列表》。
