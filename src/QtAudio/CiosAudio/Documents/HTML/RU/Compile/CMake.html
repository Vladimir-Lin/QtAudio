編譯：CMake

「CIOS音訊核心」使用CMake作為編譯工具，在您開始編譯「CIOS音訊核心」之前，您需要前往CMake官方網站，下載您的平台版本。

CMake官方網站位址：http://www.cmake.org


目標平台為ARM

如果你的目標平台是Android、iPhone OS或是Banana Pi，那麼一般而言您的CPU應當是ARMv7。

ARM的版本通常使用Cross Compiler，因此您需要選擇您的編譯主機的CMake版本。

Windows平台

Windows平台上，CMake提供了兩種方式安裝，一種是Installer，另一種是zip檔，您可以依據自己的習慣選用合適自己的安裝方式。

如果您使用zip檔，只要將CMake解開，擺到自己慣用的目錄，並且將路徑設定到環境變數上即可。

自行編譯CMake基本上是沒有必要的。

Mac OS X平台

Mac OS X平台選擇《Mac OSX 64/32-bit Universal》的dmg檔即可，下載完成後，依Mac OS X的安裝方式即可。

Linux平台

Linux平台發行版本較多，相依性較為複雜，建議您使用發行版本本身的套件，盡可能不要自行編譯。

Fedora

sudo yum install cmake


Debian

sudo apt-get install cmake


Ubuntu

sudo apt-get install cmake


FreeBSD平台

pkg install cmake
