Windows Driver Model - Kernel Streaming


微軟官方文件網址：http://msdn.microsoft.com/en-us/library/windows/hardware/ff560842(v=vs.85).aspx



啟用WDMKS模組

定義

如果您想要啟用Windows Driver Model - Kernel Streaming模組，您的C++編譯必須定義以下：

ENABLE_HOST_WDMKS

程式原始檔

C++檔案必須包含以下兩個：

  1. CiosAudio/Platforms/Windows/CaWdmks.hpp
  2. CiosAudio/Platforms/Windows/CaWdmks.cpp

Qt

如果您使用Qt，在以下設定檔內：

CiosAudio/Platforms/Windows/HostAPIs.pri

將以下設定打開即可：

HostAPIs += WDMKS


鏈結

Windows Driver Model - Kernel Streaming必須與setupapi.lib鏈結，通常這個檔案應該不會遺失。
