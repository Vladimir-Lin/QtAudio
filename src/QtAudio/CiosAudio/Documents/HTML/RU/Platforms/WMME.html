Windows Multimedia Extensions

Windows Multimedia Extensions或稱為MME API(WinMM)是Windows最早的標準化音訊介面，目前大部分已經不再使用，為了向前相容，大部分的Windows系統，依然存在 WinMM.DLL的檔案，期使較舊的程式依然可以運作。目前其文件在微軟官方網站上，已經不太容易找到。

啟用WMME模組

定義

如果您想要啟用Windows Multimedia Extensions模組，您的C++編譯必須定義以下：

ENABLE_HOST_WMME

程式原始檔

C++檔案必須包含以下兩個：

  1. CiosAudio/Platforms/Windows/CaWmme.hpp
  2. CiosAudio/Platforms/Windows/CaWmme.cpp

Qt

如果您使用Qt，在以下設定檔內：

CiosAudio/Platforms/Windows/HostAPIs.pri

將以下設定打開即可：

HostAPIs += MME


鏈結

Windows Multimedia Extensions必須與winmm.lib鏈結，通常這個檔案都應該不會遺失。
