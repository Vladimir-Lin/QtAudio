AudioScience HPI API


AudioScience HPI SDK的官方網站位址：http://www.audioscience.com/internet/download/sdk/sdk.htm



啟用ASIHPI模組

定義

如果您想要啟用AudioScience HPI API模組，您的C++編譯必須定義以下：

ENABLE_HOST_ASIHPI

程式原始檔

C++檔案必須包含以下兩個：

  1. CiosAudio/Platforms/Linux/CaASIHPI.hpp
  2. CiosAudio/Platforms/Linux/CaASIHPI.cpp

Qt

如果您使用Qt，在以下設定檔內：

CiosAudio/Platforms/Linux/HostAPIs.pri

將以下設定打開即可：

HostAPIs += ASIHPI
 

鏈結

AudioScience HPI API需要與以下函式庫鏈結：

  1. libhpi.a
  2. libhpimux.a
  3. libhpiupd.a
