开发版本（Release Candidate）

目前的Release Candidate（2015/02/13）版已经实作完整，所有音讯驱动介面均可成功编译。

已经确认可正常播放及录音的音讯驱动介面：
  • Windows Direct Sound（可录音、播放、输出音量控制，稳定性甚高。）
  • Windows Multimedia Extensions（可录音、播放、输出音量控制，稳定性甚高。）
  • Windows Audio Session API（可录音、播放、音量控制、重新取样，稳定性甚高。）
  • Windows Driver Model - Kernel Streaming（可录音、播放，稳定性甚高。）
  • Advanced Linux Sound Architecture（可录音、播放，稳定。）
  • JACK Audio Connection Kit（可录音、播放，稳定。）
  • Open Sound System（可录音、播放、音量控制，稳定性尚可。）
  • Apple Core Audio（可录音、播放、输出音量控制， 稳定性甚高。）
  • FFmpeg（可压缩、解压缩、重新取样，稳定性甚高。）

已经测试的列表可以在这裡找到《音讯驱动介面支持状态列表》。
