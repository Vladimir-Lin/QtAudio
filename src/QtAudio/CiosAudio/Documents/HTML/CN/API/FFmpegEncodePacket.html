资料结构：FFmpegEncodePacket

声道资料记录区块

类型 成员 说明
unsigned long size 资料结构封包大小。
CaHostApiTypeId hostApiType 音讯驱动介面类型。
unsigned long version 结构版本。
AVCodecContext * AudioCtx FFmpeg音讯内容。
AVCodec * AudioCodec FFmpeg音讯编码解码器。
AVStream * AudioStream FFmpeg音讯串流。
struct SwsContext * ConvertCtx FFmpeg重新取样内容。
SwrContext * Resample FFmpeg实际重新取样。

C/C++定义


 

typedef struct FFmpegEncodePacket  {
  unsigned long        size        ;
  CaHostApiTypeId      hostApiType ;
  unsigned long        version     ;
  AVCodecContext     * AudioCtx    ;
  AVCodec            * AudioCodec  ;
  AVStream           * AudioStream ;
  struct SwsContext  * ConvertCtx  ;
  SwrContext         * Resample    ;
}              FFmpegEncodePacket  ;
