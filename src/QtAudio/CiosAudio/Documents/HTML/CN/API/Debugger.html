类别：Debugger

除错机能

类型 使用 成员 说明
建构子 public Debugger(void)
解构子 public ~Debugger(void)
函式 public void printf (const char * format,...) 输出除错讯息。
函式 public virtual const char * Error (CaError errorCode) 错误码讯息。
函式 public virtual const char * lastError (void) 最后一个错误讯息。

建议

如果您是在图形介面下，那麽您应该要自行实作继承一个Debugger类别。

C/C++定义


class Debugger
{
  public:

    explicit             Debugger  (void) ;
    virtual             ~Debugger  (void) ;

    virtual void         printf    (const char * format,...) ;
    virtual const char * Error     (CaError errorCode) ;

    virtual const char * lastError (void) ;

};

extern Debugger * globalDebugger ;
