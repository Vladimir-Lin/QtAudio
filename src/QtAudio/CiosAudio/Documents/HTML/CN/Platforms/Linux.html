Linux平台的支持


如果你做了测试，并且希望能够将测试状况回报，请邮寄至linfoxman@users.sourceforge.net。


Linux平台下，目前预定支持的音讯驱动介面如下：

  • Advanced Linux Sound Architecture
  • Open Sound System
  • JACK Audio Connection Kit

预定的发行版本支持

  • Fedora
  • Ubuntu
  • Debian
  • Arch Linux
  • Scientific Linux
  • Mint
  • openSUSE
  • Gentoo
  • Slackware
  • Mandriva
  • ALT Linux

预定的支持的设备

  • x64
  • x86
  • Banana Pi ARMv7/32
