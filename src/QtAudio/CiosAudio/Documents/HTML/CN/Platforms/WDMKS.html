Windows Driver Model - Kernel Streaming


微软官方文件网址：http://msdn.microsoft.com/en-us/library/windows/hardware/ff560842(v=vs.85).aspx



启用WDMKS模组

定义

如果您想要启用Windows Driver Model - Kernel Streaming模组，您的C++编译必须定义以下：

ENABLE_HOST_WDMKS

程序原始档

C++文件必须包含以下两个：

  1. CiosAudio/Platforms/Windows/CaWdmks.hpp
  2. CiosAudio/Platforms/Windows/CaWdmks.cpp

Qt

如果您使用Qt，在以下设定档内：

CiosAudio/Platforms/Windows/HostAPIs.pri

将以下设定打开即可：

HostAPIs += WDMKS


链结

Windows Driver Model - Kernel Streaming必须与setupapi.lib链结，通常这个文件应该不会遗失。
