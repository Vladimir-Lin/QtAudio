AudioScience HPI API


AudioScience HPI SDK的官方网站位址：http://www.audioscience.com/internet/download/sdk/sdk.htm



启用ASIHPI模组

定义

如果您想要启用AudioScience HPI API模组，您的C++编译必须定义以下：

ENABLE_HOST_ASIHPI

程序原始档

C++文件必须包含以下两个：

  1. CiosAudio/Platforms/Linux/CaASIHPI.hpp
  2. CiosAudio/Platforms/Linux/CaASIHPI.cpp

Qt

如果您使用Qt，在以下设定档内：

CiosAudio/Platforms/Linux/HostAPIs.pri

将以下设定打开即可：

HostAPIs += ASIHPI
 

链结

AudioScience HPI API需要与以下函式库链结：

  1. libhpi.a
  2. libhpimux.a
  3. libhpiupd.a
