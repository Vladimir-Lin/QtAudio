Windows Audio Session API


微软官方文件网址：http://msdn.microsoft.com/en-us/library/windows/desktop/dd371455(v=vs.85).aspx



启用WASAPI模组

定义

如果您想要启用Windows Audio Session API模组，您的C++编译必须定义以下：

ENABLE_HOST_WASAPI

程序原始档

C++文件必须包含以下两个：

  1. CiosAudio/Platforms/Windows/CaWaSAPI.hpp
  2. CiosAudio/Platforms/Windows/CaWaSAPI.cpp

Qt

如果您使用Qt，在以下设定档内：

CiosAudio/Platforms/Windows/HostAPIs.pri

将以下设定打开即可：

HostAPIs += WaSAPI


链结

Windows Audio Session API，採用动态载入DLL，不需要与任何函式库链结，但是系统当中必须在路径当中可以找到avrt.dll。
