Windows Multimedia Extensions

Windows Multimedia Extensions或称为MME API(WinMM)是Windows最早的标准化音讯介面，目前大部分已经不再使用，为了向前相容，大部分的Windows系统，依然存在 WinMM.DLL的文件，期使较旧的程序依然可以运作。目前其文件在微软官方网站上，已经不太容易找到。

启用WMME模组

定义

如果您想要启用Windows Multimedia Extensions模组，您的C++编译必须定义以下：

ENABLE_HOST_WMME

程序原始档

C++文件必须包含以下两个：

  1. CiosAudio/Platforms/Windows/CaWmme.hpp
  2. CiosAudio/Platforms/Windows/CaWmme.cpp

Qt

如果您使用Qt，在以下设定档内：

CiosAudio/Platforms/Windows/HostAPIs.pri

将以下设定打开即可：

HostAPIs += MME


链结

Windows Multimedia Extensions必须与winmm.lib链结，通常这个文件都应该不会遗失。
