编译：CMake

「CIOS音讯核心」使用CMake作为编译工具，在您开始编译「CIOS音讯核心」之前，您需要前往CMake官方网站，下载您的平台版本。

CMake官方网站位址：http://www.cmake.org


目标平台为ARM

如果你的目标平台是Android、iPhone OS或是Banana Pi，那麽一般而言您的CPU应当是ARMv7。

ARM的版本通常使用Cross Compiler，因此您需要选择您的编译主机的CMake版本。

Windows平台

Windows平台上，CMake提供了两种方式安装，一种是Installer，另一种是zip档，您可以依据自己的习惯选用合适自己的安装方式。

如果您使用zip档，只要将CMake解开，摆到自己惯用的目录，并且将路径设定到环境变数上即可。

自行编译CMake基本上是没有必要的。

Mac OS X平台

Mac OS X平台选择《Mac OSX 64/32-bit Universal》的dmg文件即可，下载完成后，依Mac OS X的安装方式即可。

Linux平台

Linux平台发行版本较多，相依性较为複杂，建议您使用发行版本本身的套件，尽可能不要自行编译。

Fedora

sudo yum install cmake


Debian

sudo apt-get install cmake


Ubuntu

sudo apt-get install cmake


FreeBSD平台

pkg install cmake
