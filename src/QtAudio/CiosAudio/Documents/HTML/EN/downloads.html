Release Candidate

Current Release Candidate version(2015/02/13) has already completed the major implementation. All host APIs can be compiled on each platforms.

A comfirmed list of Host APIs that can play and record:
  • Windows Direct Sound (playback, record and volume control, highly stable)
  • Windows Multimedia Extensions (playback, record and volume control, highly stable)
  • Windows Audio Session API (playback, record, volume control and resampling, highly stable)
  • Windows Driver Model - Kernel Streaming (playback and record, highly stable)
  • Advanced Linux Sound Architecture (playback and record, stable)
  • JACK Audio Connection Kit (playback and record, stable)
  • Open Sound System (playback, record and volume control, stable)
  • Apple Core Audio (playback, record and volume control, highly stable)
  • FFmpeg (playback, record and resampling, highly stable)

A tested support list can be found here (Host APIs support status).
