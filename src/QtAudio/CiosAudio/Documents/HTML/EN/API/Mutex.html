類別：Mutex

互斥鎖。

類型 使用 成員 說明
建構子 public Mutex(void)
解構子 public ~Mutex(void)
函式 public void setMutexType (MutexType MT) 設定互鎖類型。
函式 public int lock (void) 鎖住。
函式 public int unlock (void) 解鎖。
函式 public int locked (void) 判斷是否已經被鎖住。
函式 public int tryLock (void); 嘗試鎖住。
函式 protected void releaseMutex (void) 釋放內部配置的物件。
資料 private mutex 互斥鎖。
資料 private int Locked 是否被鎖住。

建議

如果您自己已經有習慣使用的互斥鎖，那麼盡可能不要使用這裡提供的互斥鎖。

C++定義

class Mutex 
{
  public:

    enum MutexType    {
      MutexDefault    ,
      MutexNormal     ,
      MutexErrorCheck ,
      MutexRecursive
    }                 ;

    explicit  Mutex        (void);
    virtual ~ Mutex        (void);

    void      setMutexType (MutexType MT);

    int       lock         (void);
    int       unlock       (void);
    int       locked       (void);
    int       tryLock      (void);

  protected:

    void      releaseMutex (void);

  private:

    #if defined(WIN32) || defined(_WIN32)
    CRITICAL_SECTION mutex  ;
    int              Locked ;
    #else
    pthread_mutex_t  mutex;
    #endif

};
