類別：Debugger

除錯機能

類型 使用 成員 說明
建構子 public Debugger(void)
解構子 public ~Debugger(void)
函式 public void printf (const char * format,...) 輸出除錯訊息。
函式 public virtual const char * Error (CaError errorCode) 錯誤碼訊息。
函式 public virtual const char * lastError (void) 最後一個錯誤訊息。

建議

如果您是在圖形介面下，那麼您應該要自行實作繼承一個Debugger類別。

C/C++定義


class Debugger
{
  public:

    explicit             Debugger  (void) ;
    virtual             ~Debugger  (void) ;

    virtual void         printf    (const char * format,...) ;
    virtual const char * Error     (CaError errorCode) ;

    virtual const char * lastError (void) ;

};

extern Debugger * globalDebugger ;
