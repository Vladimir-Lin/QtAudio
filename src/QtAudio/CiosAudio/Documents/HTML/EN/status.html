Development status


CIOS Audio Core is currently extracted from CIOS(Collective Intelligence Operating System) into an indepentant project. Early development has been completed. However, to separate some dependency and reduce the complexity, it requires some integration and re-organized works. So far, the works result in CIOS Audio Core at Beta version. The version number follows CIOS version.

Current version is CIOS Audio Core 1.5.11 Beta.

Stable Host APIs are:
  1. Direct Sound
  2. Windows Multimedia Extensions
  3. Windows Audio Session API
  4. Windows Driver Model - Kernel Streaming
  5. Apple Core Audio
  6. FFmpeg
