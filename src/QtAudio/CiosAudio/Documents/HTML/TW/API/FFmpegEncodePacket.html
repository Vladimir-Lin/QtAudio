資料結構：FFmpegEncodePacket

聲道資料記錄區塊

類型 成員 說明
unsigned long size 資料結構封包大小。
CaHostApiTypeId hostApiType 音訊驅動介面類型。
unsigned long version 結構版本。
AVCodecContext * AudioCtx FFmpeg音訊內容。
AVCodec * AudioCodec FFmpeg音訊編碼解碼器。
AVStream * AudioStream FFmpeg音訊串流。
struct SwsContext * ConvertCtx FFmpeg重新取樣內容。
SwrContext * Resample FFmpeg實際重新取樣。

C/C++定義


 

typedef struct FFmpegEncodePacket  {
  unsigned long        size        ;
  CaHostApiTypeId      hostApiType ;
  unsigned long        version     ;
  AVCodecContext     * AudioCtx    ;
  AVCodec            * AudioCodec  ;
  AVStream           * AudioStream ;
  struct SwsContext  * ConvertCtx  ;
  SwrContext         * Resample    ;
}              FFmpegEncodePacket  ;
