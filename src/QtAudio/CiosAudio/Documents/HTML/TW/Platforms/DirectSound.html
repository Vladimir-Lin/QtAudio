Windows Direct Sound

DirectSound是微軟所開發DirectX的組件之一，在Windows平台上，目前是使用最頻繁的一個。CIOS 音訊核心在Windows平台上，內定使用Direct Sound音訊驅動介面。

官方程式設計參考網址：http://msdn.microsoft.com/en-us/library/windows/desktop/ee416964(v=vs.85).aspx

啟用Direct Sound模組

定義

如果您想要啟用Direct Sound模組，您的C++編譯必須定義以下：

ENABLE_HOST_DIRECTSOUND

程式原始檔

C++檔案必須包含以下兩個：

  1. CiosAudio/Platforms/Windows/CaDirectSound.hpp
  2. CiosAudio/Platforms/Windows/CaDirectSound.cpp

Qt

如果您使用Qt，在以下設定檔內：

CiosAudio/Platforms/Windows/HostAPIs.pri

將以下設定打開即可：

HostAPIs += DirectSound

鏈結

Windows Direct Sound必須與dsound.lib鏈結，如果您的系統當中沒有dsound.lib，那 麼您必須到微軟官方網站下載Direct X SDK來安裝。
