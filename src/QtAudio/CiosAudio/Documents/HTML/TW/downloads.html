開發版本（Release Candidate）

目前的Release Candidate（2015/02/13）版已經實作完整，所有音訊驅動介面均可成功編譯。

已經確認可正常播放及錄音的音訊驅動介面：
  • Windows Direct Sound（可錄音、播放、輸出音量控制，穩定性甚高。）
  • Windows Multimedia Extensions（可錄音、播放、輸出音量控制，穩定性甚高。）
  • Windows Audio Session API（可錄音、播放、音量控制、重新取樣，穩定性甚高。）
  • Windows Driver Model - Kernel Streaming（可錄音、播放，穩定性甚高。）
  • Advanced Linux Sound Architecture（可錄音、播放，穩定。）
  • JACK Audio Connection Kit（可錄音、播放，穩定。）
  • Open Sound System（可錄音、播放、音量控制，穩定性尚可。）
  • Apple Core Audio（可錄音、播放、輸出音量控制， 穩定性甚高。）
  • FFmpeg（可壓縮、解壓縮、重新取樣，穩定性甚高。）

已經測試的列表可以在這裡找到《音訊驅動介面支援狀態列表》。
