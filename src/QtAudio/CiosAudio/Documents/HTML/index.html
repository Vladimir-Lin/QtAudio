CIOS Audio Core is a cross-platform, open-source and free, embeddable C++ audio I/O library. It lets you write simple audio
programs in C++ that will compile and run on many platforms including Windows, Macintosh OS X, Unix, Android and
iPhone OS. It is intended to be the audio subsystem of a Robot Operating System designed by the same author.
