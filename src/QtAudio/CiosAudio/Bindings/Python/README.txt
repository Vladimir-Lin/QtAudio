Python binding CIOS Audio Core

Tools to convert C++ into Python modules:

SWIG - Simplified Wrapper and Interface Generator

http://www.swig.org

Boost.Python - this is what we normally use around here - pretty well structured

http://www.boost.org/doc/libs/1_57_0/libs/python/doc

Cython - C-Extensions for Python

http://cython.org

SIP - Riverbank SIP Python extension module

http://www.riverbankcomputing.com/software/sip/intro

PyBindGen - A tool to generate Python bindings for C/C++ code.

https://code.google.com/p/pybindgen

ECS:Python - Embedded C++ Scripting with Python

https://sourceforge.net/projects/ecspython

PyCXX - C++ facilities to make it easier to write Python extensions

http://cxx.sourceforge.net
